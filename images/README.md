# 教程图片文件夹

在此文件夹中放置教程步骤的配图。

## 使用方法

1. 将您的图片文件放入此文件夹
2. 推荐命名方式：
   - `step1.png` - 步骤1的图片
   - `step2.png` - 步骤2的图片
   - `step3.png` - 步骤3的图片
   - `step4.png` - 步骤4的图片
   - `step5.png` - 步骤5的图片

3. 在 `index.html` 中找到对应的 `<div class="step-image">` 标签
4. 在其中添加图片，例如：
   ```html
   <div class="step-image">
       <img src="images/step1.png" alt="获取二维码">
   </div>
   ```

## 图片建议

- 格式：PNG、JPG、WebP
- 宽度：建议 800-1200px
- 文件大小：建议每张 < 500KB
- 清晰度：保证截图清晰可读
